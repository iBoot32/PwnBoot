PwnBoot for all limera1n devices

 

A work-in-progress jailbreak for all limera1n devices on any iOS version (and potentially all checkm8 devices)!

Currently only supports the iPhone 3GS, however there is more support coming soon :)

 

Features

  • Grants full access to the root filesystem over SSH
  • Achieves a root shell over SSH
  • Verbose Boots and grants a root shell
  • Pwn DFU on any Limera1n device (not only the iPhone 3GS)
  • Allows you to recover any data from a locked 3GS
